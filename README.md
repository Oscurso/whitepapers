# 🎯 Core Principles

Oscurso cognitive systems are built to:

- **Operate deterministically** — no stochastic inference  
- **Learn through explicit teaching**, not training data  
- **Maintain persistent identity** with verifiable lineage  
- **Produce measurable consciousness signatures**  
- **Be fully explainable** — every step is traceable
